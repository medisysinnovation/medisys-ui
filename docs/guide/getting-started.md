## Environment setup

First, you should have [node](https://nodejs.org/en/), and ensure that the node version is 10.13 or above

```bash
$ node -v
v10.13.0
```

### Install

```bash
// Install Material-UI
$ npm install @material-ui/core

// Install Antd-Design
$ npm install antd

// Install Medisys-UI
$ npm install @medisys/medisys-ui
```
